# Step by Step

1. **Getting Started**
    - Connect to a Wi-Fi network. 
    - Go back to start.
2. **Turn on TalkBack shortcut**
    - Hold Volume+ & Volume- 
    - Turn on TalkBack shortcut ➡️ Turn on
3. **Switch to TalkBack braille keyboard** 
    - Choose languge ➡️ English (US) 
    - Choose your region ➡️ India (Important)
    - Choose keyboard ➡️ TalkBack braille keyboard 
4. **Enable accessibility menu**     
    - Agree to terms and conditions ➡️ Next
    - Add network ➡️ Tap Network name input area ➡️ Tun on accessibiility settings.
    - Accessibility menu ➡️ Toggle Accessibility menu shortcut On. 
5. **Opening Settings App** 
    - Tap floating yellow button appear on the bottom right
    - Google Assistant ➡️ Tap Keyboad icon ⌨️ ➡️ Next keyboard
    - Type "Open Settings"
6. **Install Activity Launcher**
    - In Settings app ➡️ Apps ➡️ Manage apps 
    - Find apps (Only appear on India region)
    - In Xiaomi GetApps store ➡️ Search & Install Activity Launcher.
7. **Turn on Second space**
    - Open Activity Launcher ➡️ Search "Second Space"
    - Tap on Second space with "com.miui.securityspace.settings.SEcondSpaceSettingsActivity"
    - Turn on Second space ➡️ Using a shortcut ➡️ Skip
8. **Reboot & Factory Reset**
    - Reboot the device. After reboot, do a factory reset in settings.

> [!NOTE]
> Tested on Redmi 10C MiUi 13 (23 June 2025)
